Fresh from Oven
Machine Learning Assisted Chimera and Solitary States in Networks
Here, using supervised machine learning, we predict (a) the precise value of delay which is sufficient for engineering chimera and solitary states for a given set of system's parameters, as well as (b) the intensity of incoherence for such engineered states. Ergo, using few initial data points we generate a machine learning model which can then create a more refined phase plot as well as by including new parameter values.
